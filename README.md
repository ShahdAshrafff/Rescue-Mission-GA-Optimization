# Rescue Mission Path Optimization 🤖🆘

A hybrid AI pipeline that combines **Breadth-First Search (BFS)** for obstacle avoidance and **Genetic Algorithms (GA)** for global path optimization.

## Features
- **BFS:** Calculates the true distance matrix avoiding walls.
- **Genetic Algorithm:** Optimizes the sequence of 5 victims using:
  - **Rank-based Selection**
  - **Ordered Crossover**
  - **Swap Mutation**

## Results
- **Optimal Distance:** 45.0 steps.
- **Language:** Python (NumPy, Matplotlib).
